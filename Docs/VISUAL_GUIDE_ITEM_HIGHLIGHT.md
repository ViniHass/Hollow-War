## ?? Workflow Visual

```
PASSO 1: PREPARAR
?? Criar/Organizar prefabs de itens
?? Ter SpriteRenderer em cada item

PASSO 2: APLICAR
?? Abrir ferramenta (Tools ? Item Highlight)
?? Configurar caminho da pasta
?? Clicar em "Aplicar a Todos"

PASSO 3: AJUSTAR (Opcional)
?? Selecionar prefab específico
?? Ajustar cor/intensidade
?? Salvar mudanças

PASSO 4: TESTAR
?? Criar cena de teste
?? Arrastar prefabs para cena
?? Entrar em Play Mode
```
